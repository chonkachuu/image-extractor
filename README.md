# Universal Image Extractor

A powerful, single-file HTML application to extract, filter, and preview images from any source code or list of URLs.

## Features

- **Extract Images**: Paste raw HTML source code or text containing image URLs to extract them instantly.
- **Filtering**: Filter images by minimum size (width or height).
- **Sorting**: Sort images by original order, resolution, filename, file type, or aspect ratio.
- **Preview**: Grid view of extracted images with details like resolution, orientation, and format.
- **Actions**:
  - Open image in a new tab.
  - Download image directly.
  - Copy individual image URL.
  - Copy all extracted image URLs at once.
- **Format Support**: Supports JPG, PNG, WEBP, GIF, AVIF, SVG, and more.
- **No Installation Required**: Just open the HTML file in any modern web browser.

## Usage

1. **Download** or **Clone** this repository.
2. **Open** `index.html` in your web browser (Chrome, Firefox, Edge, Safari, etc.).
3. **Paste** your content (HTML source code or text with links) into the text area.
4. Click **Process Content**.
5. Use the toolbar to filter by size or sort the results.

## Dependencies

- **Tailwind CSS**: Used for styling, loaded via CDN (requires internet connection).
